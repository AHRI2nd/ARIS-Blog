Lyrical Sync의 AI 자동 싱크 기능은 음성 인식(CTC Forced Aligner)을 활용해 가사 텍스트에 타임스탬프를 자동으로 부여합니다.  
선택적으로 Demucs 보컬 분리를 전처리에 사용해 정확도를 높일 수 있습니다.

---

## 시스템 요구사항

| 항목 | 최소 사양 |
|------|-----------|
| OS | macOS 12+, Windows 10+ |
| RAM | 8 GB (16 GB 권장) |
| 저장 공간 | 필수 모델만: 약 3 GB / 전체: 약 5 GB |
| 인터넷 | 최초 설치 시 필요 |

> [!NOTE]
> **GPU 가속**: 현재 버전은 CPU 추론을 사용합니다. 성능은 CPU 속도에 따라 다릅니다.

---

## 1단계 — Python 환경 설치

AI 기능은 별도의 Python 설치 없이 앱 내장 Python(CPython 3.11)을 사용합니다.

1. 앱 좌측 하단의 **⚙ 설정** 버튼을 클릭합니다.
2. **AI 환경** 탭을 선택합니다.
3. **"Python 다운로드"** 버튼을 클릭합니다.
   - macOS: `python-3.11.10-macos-...` 배포판이 다운로드됩니다.
   - Windows: `python-3.11.10-windows-...` 배포판이 다운로드됩니다.
4. 다운로드 완료 후 **"패키지 설치"** 버튼을 클릭합니다.
   - `torch`, `unidecode`, `demucs`, `ctc-forced-aligner` 등이 설치됩니다.
   - Windows에서는 `ctc-forced-aligner`를 소스에서 빌드합니다 (10~20분 소요).
5. 설치 로그에 **"설치 완료"** 메시지가 표시되면 완료입니다.

> [!WARNING]
> **Windows 주의**: Visual Studio Build Tools(C++ 빌드 도구)가 없으면 소스 빌드가 실패할 수 있습니다. 오류 발생 시 [Visual Studio Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)를 먼저 설치하세요.

---

## 2단계 — 모델 다운로드

1. 설정의 **AI 모델** 탭을 선택합니다.
2. 아래 표를 참고해 필요한 모델을 다운로드합니다.

| 모델 | 용도 | 크기 | 필수 여부 |
|------|------|------|-----------|
| **ctc-mms-300m** | 타임스탬프 강제 정렬 (1,100개 언어 지원) | 1.2 GB | ✅ 필수 |
| demucs-htdemucs | 보컬/반주 분리 전처리 | 83 MB | 선택 |
| wav2vec2-base-960h | 영어 ASR 보조 모델 | 380 MB | 선택 |

3. 모델 파일은 기본적으로 앱 데이터 디렉터리에 저장됩니다.  
   커스텀 경로를 지정하려면 **"모델 저장 경로"** 항목에서 변경할 수 있습니다.

> [!TIP]
> 처음에는 ctc-mms-300m만 설치해 사용해보세요. Demucs는 반주가 많은 곡에서 정확도를 높여주지만, 처리 시간이 추가됩니다.

---

## 3단계 — AI 자동 싱크 실행

1. 오디오 파일을 엽니다 (MP3, WAV, FLAC, M4A 등).
2. LRC 에디터에 가사를 입력합니다.
3. 에디터 상단의 **"AI 자동 싱크"** 버튼을 클릭합니다.
4. 언어를 선택합니다.
   - 한국어: `kor`
   - 영어: `eng`
   - 일본어: `jpn`
   - 그 외 1,100개 언어 지원 (ISO 639-3 코드)
5. **"실행"** 버튼을 클릭합니다.

### 진행 상태

| 상태 | 설명 |
|------|------|
| 모델 로딩 | ctc-mms-300m 모델을 메모리에 올리는 중 |
| 보컬 분리 | Demucs로 보컬 트랙 추출 중 (Demucs 설치 시) |
| 정렬 분석 | 가사와 오디오를 매칭하는 중 |
| 완료 | 타임스탬프 생성 완료 |

---

## 4단계 — 결과 확인 및 보정

- 각 가사 줄 옆에 **신뢰도 배지**가 표시됩니다 (0~100%).
- 신뢰도가 낮은 줄은 배지가 빨간색으로 표시됩니다.
- 보정 방법:
  - 해당 줄을 클릭해 활성화한 뒤 재생 중 **Space** 키로 수동 스탬핑
  - 타임스탬프 버튼을 직접 클릭해 현재 재생 위치로 덮어쓰기
- 수동 스탬핑한 줄은 신뢰도 배지가 자동으로 제거됩니다.
- 모든 확인이 끝나면 **"초안 지우기"** 버튼으로 신뢰도 배지를 전체 삭제할 수 있습니다.

## 자주 묻는 질문

**Q. 정렬이 틀린 줄이 많아요.**  
A. 가사에 오타나 불필요한 기호(괄호, 코러스 표시 등)가 없는지 확인하세요. 보컬이 섞인 곡은 Demucs 설치 후 재시도하면 정확도가 올라갑니다.

**Q. "Python이 준비되지 않았습니다" 오류가 떠요.**  
A. 설정 → AI 환경에서 Python 설치 상태를 확인하세요. 설치가 완료됐는데도 오류가 지속되면 앱을 재시작해보세요.

**Q. macOS에서 "손상된 파일" 경고가 떠요.**  
A. 앱이 자동으로 quarantine 속성을 제거하지만, 실패할 경우 터미널에서 아래 명령을 실행하세요.

```bash
xattr -rd com.apple.quarantine ~/Library/Application\ Support/lyrical-sync/python
```

**Q. Windows에서 패키지 설치가 실패해요.**  
A. Visual Studio Build Tools의 "C++를 사용한 데스크톱 개발" 워크로드를 설치한 후 다시 시도하세요.
